Oracle 19c 로그를 /logs 아래로 이동시키기 위한 파라미터 설정표입니다.

## DB 파라미터 설정

| 파라미터명 | 변경값 | 적용범위 | 설명 |
|---|---|---|---|
| diagnostic_dest | '/logs' | SPFILE | ADR 루트 (alert log 포함) |
| audit_file_dest | '/logs/audit' | SPFILE | Audit 로그 디렉토리 |
| core_dump_dest | '/logs/cdump' | SPFILE | Core dump 파일 |
| background_dump_dest | '/logs/bdump' | SPFILE | Background process trace |
| user_dump_dest | '/logs/udump' | SPFILE | User process trace |

## Listener 파라미터 설정 (listener.ora)

| 파라미터명 | 변경값 | 설명 |
|---|---|---|
| LOG_DIRECTORY_LISTENER | /logs/listener | Listener 로그 디렉토리 |
| LOG_FILE_LISTENER | listener.log | Listener 로그 파일명 |
| TRACE_DIRECTORY_LISTENER | /logs/listener/trace | Listener trace 디렉토리 |
| TRACE_FILE_LISTENER | listener.trc | Listener trace 파일명 |

## 실제 로그 저장 위치

| 로그 유형 | 저장 경로 |
|---|---|
| Alert Log | /logs/diag/rdbms/[DB_NAME]/[INSTANCE_NAME]/trace/alert_[INSTANCE_NAME].log |
| Audit Log | /logs/audit/*.aud |
| Listener Log | /logs/listener/listener.log |
| Core Dump | /logs/cdump/*.dmp |
| Background Trace | /logs/bdump/*.trc |
| User Trace | /logs/udump/*.trc |

## SQL 명령어

```sql
ALTER SYSTEM SET diagnostic_dest='/logs' SCOPE=SPFILE;
ALTER SYSTEM SET audit_file_dest='/logs/audit' SCOPE=SPFILE;
ALTER SYSTEM SET core_dump_dest='/logs/cdump' SCOPE=SPFILE;
ALTER SYSTEM SET background_dump_dest='/logs/bdump' SCOPE=SPFILE;
ALTER SYSTEM SET user_dump_dest='/logs/udump' SCOPE=SPFILE;
```

**요약**: 총 5개 DB 파라미터와 4개 Listener 파라미터를 변경하여 모든 Oracle 로그를 /logs 아래로 이동시킬 수 있습니다.
